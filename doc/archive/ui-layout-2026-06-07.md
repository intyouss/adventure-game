## 交互模式

### Mode.NORMAL（默认）
- BattleArea: visible ✓
- EquipmentArea: visible ✓
- ChestArea: visible ✓
- SkillInventoryPanel: hidden
- ShopPanel: hidden
- LeaderboardPanel: hidden

### Mode.SKILL_INVENTORY（技能仓库）
- 触发: 点击底部"📋 技能仓库"
- BattleArea: visible ✓
- EquipmentArea: hidden
- ChestArea: hidden
- SkillInventoryPanel: visible ✓（6列网格，24个/页）
- ShopPanel: hidden
- 高亮: 技能仓库按钮变黄

### Mode.SHOP（商店）
- 触发: 点击底部"🏪 商店"
- BattleArea: hidden
- EquipmentArea: hidden
- ChestArea: hidden
- SkillInventoryPanel: hidden
- ShopPanel: visible ✓（2列网格）
- 高亮: 商店按钮变黄

### 排行榜侧滑
- 触发: 点击底部"🏆"或右侧按钮
- 动画: AnimationPlayer slide_in 0.3s
- 展开后自身按钮隐藏，面板上有收起按钮
- 内容: 章节标签切换 + 排名列表（含我的排名置顶高亮）+ 加载更多

### 开箱装备对比弹窗
- 触发: 开箱获得装备时自动弹出
- 有空槽位 → 仅显示新装备属性，"保留原装备"隐藏，"装备"按钮
- 有装备 → 上下对比（新装备 vs 当前装备），"保留原装备"/"替换新装备"
- 未选装备自动分解
